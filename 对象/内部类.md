# 介绍

> 一个类的内部又完整的嵌套了另一个类接口，被嵌套的类成为内部类
>
> 是类的五大成员：属性、方法、构造器、代码块、内部类

内部类最大的特点是可以直接访问私有属性，并且可以体现类与类之间的包含关系

```java
class Outer{ // 外部类
  class Inner{ // 内部类
    
  }
}

class Other{ // 外部其他类
  
}
```



# 分类

* 定义在外部类局部位置上（比如方法内）：
  * 局部内部类（有类名）
  * 匿名内部类（没有类型，重点！！！！）
* 定义在外部类的成员位置上：
  * 成员内部类（没用static修饰）
  * 静态内部类（使用static修饰）
